- 安装过程报出的错误解决方案：
- 报错：需要重新启动计算机.必须重新启动计算机才能安装 SQL Server
- 解决办法：
在开始运行中输入: regedit
找到HKEY_LOCAL_MACHINE\SYSTEM\CurrentControlSet\Control\SessionManager 
在右边窗口找到PendingFileRenameOperations删除即可
